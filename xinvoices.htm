XInvoices

Standard Edition

Does your business need an easy Invoicing solution? XInvoices can generate PDF Invoices based simply and easily using simple XML files as input.

Features


  • Generate PDF invoices in two steps
  • Sample input and output included in web application
  • InvoiceXML® schema defined and documented
  • Control the branding of your invoices from the InvoiceXML® input file
  • Specify email details in the InvoiceXML® input file
  • Email invoices to customers directly from XInvoices
  • Both Console and Web Interfaces included in download
  • User manual included in download and embedded into web application
  • Online support included in price
