HydratedSolutions 4.0

Standard Edition

Produces navigable object maps of Visual Studio solution and project files. Supports .sln, .csproj and .wixproj files.

The information currently gathered in the current release is:
  • Global properties of the solution
  • List of projects in solution
  • Global properties of each project
  • List of source files in each project
  • List of System references that each project depends on
  • List of Project references that each project depends on
  • Basic properties of each source file


Features

  • "Hydrate" a solution in one step
  • Visual Studio-like interface
  • File System View and Solution View
  • Both Console and Web Interfaces included in download
  • User manual included in download and embedded into web application
  • Online support included in price
