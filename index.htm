  • XRayMachine - Code Generator for XML files
    Developers sometimes need to do repetitive tasks. Tools can ease this burden. XML has been and continues to be a common data format for a long time. It is likely that, like Cobol it will continue to persist in both contemporary and legacy applications for decades to come. What if there was a tool that made working with XML a breeze? XRayMachine is just that tool. XRayMachine creates an accessor class for any XML file enabling easy access to every attribute and collection contained therein.



    Features

    • One step C# accessor class generation
    • Sample input and output
    • Console and web interfaces
    • User manual embedded in web application
    • Online support included

  • HydratedSolutions - Visual Studio Solution Analyser
    Your boss needs you to understand the structure of a solution quickly but won't stump up $19,274 for a Visual Studio licence?
    HydratedSolutions to the rescue. All you need in the .sln file and the csproj files and HydratedSolutions will take care of the rest.
    HydratedSolutions will "Hydrate"® the raw text files to give you a Visual Studio-like view of the solution and how the components are coupled.



    Features

    • "Hydrate" a solution in one step
    • Visual Studio-like interface
    • File System View and Solution View
    • Both Console and Web Interfaces
    • User manual embedded in web application
    • Online support included
