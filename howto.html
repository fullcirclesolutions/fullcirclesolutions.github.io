How To Integrate Saucery

How To Integrate Saucery

Create a SauceLabs account

Go to SauceLabs and signup for a free trial account

Reference & Activate

Create a simple class library and reference Saucery2 (for NUnit2) or Saucery3 (for NUnit 3)

Get your Saucery Activation key from the download page.

Write your tests

Check out the examples.

Write Selenium Tests for your application without having to think about DesiredCapabilities at all. Integration with SauceLabs is seamless!

Execute!

Install the Sauce OnDemand plugin for Jenkins or Bamboo.

Specify the platforms you wish to run your tests on in the job configuration.

Execute your CI job and unleash the awesome!
